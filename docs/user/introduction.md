# Introduction

The **Cartographer App** provides placement information of zones of interest in (historical) documents.  
The primary use case is the identification of bounding boxes of measures in music documents.  

It is a successor of the Vertaktoid, but unlike that project, it uses **web technology** and is thus platform-independent.  

It can optionally use the **Measure Detector** for automatic recognition of measure positions, while still allowing manual correction.
